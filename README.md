# DSA (Data Structures and Algorithms) Repository

This repository contains **Java implementations of Data Structures and Algorithms** based on a typical DSA syllabus. Each folder corresponds to a topic, and the code includes **basic implementations, examples, and updated exercises**.

---

## 📂 Folder Structure

### 1️⃣ `.vscode`
Contains VS Code workspace settings (`settings.json`, `launch.json`).  
No DSA code here.

---

### 2️⃣ `LinkedList`
Includes:
- `SinglyLinkedList.java` – Singly Linked List implementation
- `DoublyLinkedList.java` – Doubly Linked List implementation
- `SinglyCircularLinkedList.java` – Singly Circular Linked List implementation
- `DoublyCircularLinkedList.java` – Doubly Circular Linked List implementation

**Description:** Implements insertion, deletion, traversal, search, and circular variations.

---

### 3️⃣ `Stack`
Includes:
- `ArrayStack.java` – Stack implemented using arrays
- `LinkedStack.java` – Stack implemented using linked list

**Description:** Implements stack operations: push, pop, peek, and demonstrates Stack ADT and evaluation of expressions.

---

### 4️⃣ `Queue`
Includes:
- `ArrayQueue.java` – Queue implemented using arrays
- `LinkedQueue.java` – Queue implemented using linked list

**Description:** Implements queue operations: enqueue, dequeue, peek, and demonstrates Queue ADT.

---

### 5️⃣ `Sorting`
Includes:
- `BubbleSort.java`
- `InsertionSort.java`
- `SelectionSort.java`
- `MergeSort.java`
- `QuickSort.java`
- `RadixSort.java`

**Description:** Implements basic and advanced sorting algorithms.  
**Note:** HeapSort can be added in future updates.

---

### 6️⃣ `Tree`
Includes:
- `BinaryTree.java` – Simple binary tree
- `BinarySearchTree.java` – BST implementation
- `Node.java`
- `TestBST.java`

**Description:** Covers tree operations such as insertion, deletion, and traversal (in-order, pre-order, post-order).  
**Future updates:** AVL Trees, Heaps, Expression Trees, Huffman, B-Trees.

---

### 7️⃣ `Graph`
**Planned / Future:**  
- `AdjacencyMatrixGraph.java`  
- `AdjacencyListGraph.java`  

**Description:** Basic graph representations and traversals (BFS, DFS).  
**Future updates:** Shortest path algorithms (Dijkstra, Floyd-Warshall), Minimum Spanning Tree (Kruskal, Prim), Topological Sort.

---

### 8️⃣ `Recursion` (Planned)
**Description:** Recursive examples like Factorial, Fibonacci, GCD, Tower of Hanoi, Direct/Tail/Nested recursion.

---

### 9️⃣ `Searching & Hashing` (Planned)
**Description:** Linear search, Binary search, Hashing, Hash tables, and collision resolution.

---

## ⚡ Notes
- Each folder corresponds to a DSA topic.
- Code includes examples and common operations for learning and practice.
- Recommended to use **Java 8+** for running these programs.
- Future updates will include advanced topics like AVL Trees, Graph Algorithms, Heap, Hash Tables, and Recursion examples.
