DSA (Data Structures and Algorithms) Repository

This repository contains Java implementations of Data Structures and Algorithms based on a typical DSA syllabus. Each folder corresponds to a topic, and the code includes basic implementations as well as examples.

📂 Folder Structure
1️⃣ .vscode

Purpose: Contains VS Code workspace settings.

Files: settings.json, launch.json

Note: No DSA code here.

2️⃣ LinkedList

Includes:

SinglyLinkedList.java – Singly Linked List implementation

DoublyLinkedList.java – Doubly Linked List implementation

SinglyCircularLinkedList.java – Singly Circular Linked List implementation

DoublyCircularLinkedList.java – Doubly Circular Linked List implementation

Description:
Implements all common operations: insertion, deletion, traversal, and search.

3️⃣ Stack

Includes:

ArrayStack.java – Stack implemented using arrays

LinkedStack.java – Stack implemented using linked list

Description:
Implements stack operations (push, pop, peek) and demonstrates Stack ADT.

4️⃣ Sorting

Includes:

BubbleSort.java

InsertionSort.java

SelectionSort.java

Description:
Implements basic sorting algorithms. Future updates may include QuickSort, MergeSort, HeapSort, and RadixSort.

5️⃣ Tree

Includes:

BinaryTree.java – Simple binary tree implementation

BinarySearchTree.java – Binary search tree implementation

Description:
Covers tree operations like insertion, deletion, traversal (in-order, pre-order, post-order).
