# DSA (Data Structures and Algorithms) Repository

This repository contains **Java implementations of Data Structures and Algorithms** based on a typical DSA syllabus. Each folder corresponds to a topic, and the code includes **basic implementations, examples, and updated exercises**.

---

## 📂 Folder Structure

### 1️⃣ `.vscode`
**Purpose:** Contains VS Code workspace settings.  
**Files:** `settings.json`, `launch.json`  
**Note:** No DSA code here.

---

### 2️⃣ `LinkedList`
**Includes:**  
- `SinglyLinkedList.java` – Singly Linked List implementation  
- `DoublyLinkedList.java` – Doubly Linked List implementation  
- `SinglyCircularLinkedList.java` – Singly Circular Linked List implementation  
- `DoublyCircularLinkedList.java` – Doubly Circular Linked List implementation  

**Description:** Implements all common operations: **insertion, deletion, traversal, search, and circular variations**.

---

### 3️⃣ `Stack`
**Includes:**  
- `ArrayStack.java` – Stack implemented using arrays  
- `LinkedStack.java` – Stack implemented using linked list  

**Description:** Implements stack operations: **push, pop, peek**, and demonstrates **Stack ADT**.

---

### 4️⃣ `Queue`
**Includes:**  
- `ArrayQueue.java` – Queue implemented using arrays  
- `LinkedQueue.java` – Queue implemented using linked list  

**Description:** Implements **queue operations**: enqueue, dequeue, peek, and demonstrates **Queue ADT**.

---

### 5️⃣ `Sorting`
**Includes:**  
- `BubbleSort.java`  
- `InsertionSort.java`  
- `SelectionSort.java`  

**Description:** Implements basic sorting algorithms. Future updates may include **QuickSort, MergeSort, HeapSort, and RadixSort**.

---

### 6️⃣ `Tree`
**Includes:**  
- `BinaryTree.java` – Simple binary tree implementation  
- `BinarySearchTree.java` – Binary search tree implementation  

**Description:** Covers tree operations like **insertion, deletion, traversal (in-order, pre-order, post-order)**.

---

### 7️⃣ `Graph` *(if added)*
**Includes:**  
- `AdjacencyMatrixGraph.java`  
- `AdjacencyListGraph.java`  

**Description:** Implements basic **graph representations and traversals**: BFS and DFS.

---

### 8️⃣ `Others / Advanced` *(future updates)*
- **InfixPostfix.java** – Conversion and evaluation of expressions  
- **Heap.java** – MaxHeap and MinHeap implementation  
- **HashTable.java** – Basic hash table implementation  

**Description:** These will include advanced DSA topics and exercises for practice.

---

## ⚡ Notes
- Each folder corresponds to a DSA topic.  
- Code includes **examples and common operations** for learning and practice.  
- Recommended to use **Java 8+** for running these programs.
